# 技术债：待重构指标清单

## 1. 概述

本清单记录了所有不符合《指标开发规范》的、需要重构的技术指标。这些指标的形态识别方法 (`get_patterns` 或 `identify_patterns`) 目前返回的是已弃用的 `List[Dict]` 或 `List[str]` 格式，存在性能和维护性问题。

**重构目标**: 将这些指标的形态识别方法修改为返回标准的布尔型 `pandas.DataFrame`。
